TUI SPREADSHEET - Edit Mode Example
====================================

┌──────────────────────────────────────────────────────────────────────────────┐
│ Edit Mode (Enter=Save, Esc=Cancel)                                           │
│ B6 | fx: =SUM(B2:B5)                                                          │
└──────────────────────────────────────────────────────────────────────────────┘
┌──────────────────────────────────────────────────────────────────────────────┐
│ Spreadsheet                                                                   │
├────┬────────────┬────────────┬────────────┬────────────┬────────────────────┤
│    │     A      │     B      │     C      │     D      │     E              │
├────┼────────────┼────────────┼────────────┼────────────┼────────────────────┤
│ 1  │ 150        │            │            │            │                    │
│ 2  │ 10         │ 100        │            │            │                    │

│ 4  │ 30         │ 300        │            │            │                    │

EDIT | Cell: B6 | Functions: =SUM(A1:A10), =AVERAGE(A1:A10), =MIN, =MAX, =COUNT


Features Demonstrated:
- Cell A1 displays the computed result (150) of =SUM(A2:A5)
- In edit mode, cell B6 shows the formula being entered: =SUM(B2:B5)
- Column/row headers for easy reference
- Cursor highlighting (shown in blue in actual TUI)
- Formula bar at top showing current cell reference and value
- Status bar at bottom with mode indicator and help text
